# Awesome bare-native

A curated list of resources, libraries, and projects for building native applications with [bare-native](https://github.com/holepunchto/bare-native).

> **Note:** Everything (except `bare-native`) listed here so far is experimental and AI-assisted — more is coming. Labels indicate status.

---

## Labels

- `experimental` — not production-ready, API may change
- `claude` — built with significant AI assistance

---

## Core

- [bare-native](https://github.com/holepunchto/bare-native) — run Bare as a native application target

---

## UI Libraries

- [sveltekit-adapter-bare](https://github.com/holepunchto/sveltekit-adapter-bare) `experimental` `claude` — SvelteKit adapter for Bare; see also [bare-svelte-demo](https://github.com/Drache93/bare-svelte-demo)
- [cellery](https://github.com/holepunchto/cellery) `experimental` — standalone low-level UI primitive library for Bare

---

## Platform Libraries

- [bare-wifi-android](https://github.com/holepunchto/bare-wifi-android) `experimental` `claude` — WiFi management bindings for Android
- [bare-navigation-android](https://github.com/holepunchto/bare-navigation-android) `experimental` `claude` — navigation stack bindings for Android
- [bare-appletv-remote](https://github.com/Drache93/bare-appletv-remote) `experimental` `claude` — Apple TV remote control bindings for Bare

---

## Demos & Experiments

- [holebooks](https://github.com/Drache93/holebooks) `claude` — P2P reading app built with SvelteKit on Bare
- [ghost-drive](https://github.com/Drache93/ghost-drive) `experimental` `claude` — P2P file drive UI built with SvelteKit on Bare
- [bare-zork](https://github.com/Drache93/bare-zork) `experimental` `claude` — text adventure built with cellery on Bare
- [bare-atv](https://github.com/Drache93/bare-atv) `claude` — Apple TV remote control app on Bare
